### The Simple PVP Toggle DataPack

As Minecraft has removed the PVP option from the server.properties file and moved it to a gamerule, this helpful datapack can change the PVP gamerule across all dimensions.

A simple way to enable and disable PVP across all dimensions on your Minecraft Server or world.

**Available Commands:**

To Enable:
/function simplepvptoggle:pvpenable

To Disable:
/function simplepvptoggle:pvpdisable
